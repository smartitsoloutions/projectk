What is Maven ?

The developers call it a build automation tool but 
it is much more than that it is a project management tool. 
Building a project means
compiling the source code running the tests which could be unit tests as well
as integration tests packaging the compiled code into jar files bundling
these jar files into a web archive or a war file deploying these wars on 
to the servers and several other tasks all these tasks can be automated using tools
such as maven and etc

To set the environmet variable for maven
go to /etc/profile.d then create a file with maven.sh then type
export M2_HOME=/opt/apache-maven
export PATH=${M2_HOME}/bin:${PATH} now save the file.
Now change file permission
chmod 777 maven.sh
Now reload the service
source /etc/profile.d/maven.sh
then check it using mvn --version
